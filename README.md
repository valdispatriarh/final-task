- ESLint
- Prettier
- Jest 
- Git
- HTML5
- CSS3
- JavaScript/TypeScript
- Vue3
- Vite

## instalation 

If you have project local in terminal use: 
``` git pull ```
Next step in terminal is use:  
```npm install```
for install dependecies. 

If you want start eslint you must use commend: 

```npm run lint``` 

If you want start test script in jest use: 

```npm run test```
